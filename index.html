API Basejump: Image Search

By Rex "SaintPeter" Schrader | Project GitHub

User Stories:
  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a /2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example usage:

https://saintpeter-image-search.herokuapp.com/search/funny%20cats
https://saintpeter-image-search.herokuapp.com/search/funny%20cats/2
https://saintpeter-image-search.herokuapp.com/recent/
